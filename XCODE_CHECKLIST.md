# Xcode Checklist - What to Verify

Once Xcode opens, check these items:

## ✅ Checklist

### 1. Signing & Capabilities
- [ ] Open project: Click "17WEEKS" (blue icon, left sidebar)
- [ ] Select target: Click "17WEEKS" under TARGETS
- [ ] Click "Signing & Capabilities" tab
- [ ] Verify:
  - [ ] "Automatically manage signing" is ✅ checked
  - [ ] Team is selected (your Apple Developer account)
  - [ ] Bundle Identifier: `com.seventeenweeks.dev`
  - [ ] No red errors/warnings

### 2. Version & Build
- [ ] Check Version: Should be `1.0` or `1.0.0`
- [ ] Check Build: Should be `1` or higher
- [ ] If needed, click "General" tab to update

### 3. Device Selection (For Archiving)
- [ ] Top toolbar: Device selector dropdown
- [ ] Select: "Any iOS Device (arm64)" or "Generic iOS Device"
- [ ] DO NOT select a simulator

### 4. Ready to Archive
- [ ] All checks above are ✅
- [ ] No red errors in project
- [ ] Ready to proceed to archiving

---

**Once everything is checked, you're ready to archive!**
